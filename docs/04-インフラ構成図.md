# インフラ構成図

## ローカル開発環境

```mermaid
graph TB
    subgraph ローカルマシン
        Uvicorn[uvicorn :8000]
        Streamlit[streamlit :8501]
        SQLite[(SQLite)]
        ChromaDB[(ChromaDB)]
    end

    subgraph 外部
        LINE[LINE Platform]
        Gemini[Gemini API]
        ngrok[ngrok]
    end

    Uvicorn --> SQLite
    Uvicorn --> ChromaDB
    Uvicorn --> Gemini
    Streamlit --> SQLite
    Streamlit --> ChromaDB
    Streamlit --> Gemini
    ngrok -->|HTTPS| Uvicorn
    LINE -->|Webhook| ngrok
```

## Cloud Run 本番環境（案A: 完全無料寄り）

```mermaid
graph TB
    subgraph ユーザー
        LINE[LINE アプリ]
        Browser[管理者ブラウザ]
    end

    subgraph Google Cloud
        subgraph Cloud Run
            API[line-rag-api<br/>FastAPI]
            Admin[line-rag-admin<br/>Streamlit]
        end

        subgraph Firestore
            ConfigCol[config コレクション]
            KnowledgeCol[knowledge コレクション]
        end

        subgraph 外部API
            Gemini[Gemini API]
        end
    end

    subgraph LINE Platform
        LP[LINE Messaging API]
    end

    LINE -->|メッセージ| LP
    LP -->|POST /webhook| API
    API --> ConfigCol
    API --> KnowledgeCol
    API --> Gemini
    API -->|返信| LP

    Browser -->|HTTPS| Admin
    Admin --> ConfigCol
    Admin --> KnowledgeCol
    Admin --> Gemini
```

## デプロイパイプライン

```mermaid
flowchart LR
    subgraph ソース
        Code[リポジトリ]
        Env[.env]
    end

    subgraph Cloud Build
        Build1[Dockerfile.api]
        Build2[Dockerfile.admin]
        Push[Artifact Registry Push]
    end

    subgraph GCP
        AR[Artifact Registry]
        CR1[Cloud Run: line-rag-api]
        CR2[Cloud Run: line-rag-admin]
    end

    Code --> Build1
    Code --> Build2
    Build1 --> Push
    Build2 --> Push
    Push --> AR
    AR --> CR1
    AR --> CR2
    Env -->|deploy.sh| CR1
    Env -->|deploy.sh| CR2
```

## ネットワーク構成

| コンポーネント | 入力 | 出力 |
|----------------|------|------|
| line-rag-api | LINE Webhook (HTTPS) | Firestore, Gemini API, LINE API |
| line-rag-admin | 管理者ブラウザ (HTTPS) | Firestore, Gemini API |
| Firestore | Cloud Run サービス | - |
| Gemini API | Cloud Run サービス | - |
