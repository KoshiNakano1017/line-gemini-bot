# システム構成図

## アプリケーション構成

```mermaid
graph TB
    subgraph クライアント
        LINE[LINE アプリ]
        Admin[管理者ブラウザ]
    end

    subgraph アプリケーション
        FastAPI[FastAPI<br/>Webhook API]
        Streamlit[Streamlit<br/>管理画面]
        
        subgraph ドメイン
            WebhookHandler[Webhook ハンドラ]
            PromptBuilder[プロンプト構築]
        end
        
        subgraph インフラ層
            ConfigStore[(ConfigStore)]
            KnowledgeStore[(KnowledgeStore)]
            GeminiClient[Gemini Client]
            LineClient[LINE API Client]
        end
    end

    subgraph 外部サービス
        LINEPlatform[LINE Platform]
        GeminiAPI[Gemini API]
    end

    subgraph ストレージ
        Firestore[(Firestore)]
        SQLite[(SQLite)]
        ChromaDB[(ChromaDB)]
    end

    LINE -->|メッセージ| LINEPlatform
    LINEPlatform -->|POST /webhook| FastAPI
    FastAPI --> WebhookHandler
    WebhookHandler --> ConfigStore
    WebhookHandler --> KnowledgeStore
    WebhookHandler --> GeminiClient
    WebhookHandler --> LineClient
    WebhookHandler --> PromptBuilder
    PromptBuilder --> GeminiClient
    LineClient -->|返信| LINEPlatform
    GeminiClient -->|生成・埋め込み| GeminiAPI

    ConfigStore --> Firestore
    ConfigStore --> SQLite
    KnowledgeStore --> Firestore
    KnowledgeStore --> ChromaDB

    Admin -->|HTTP| Streamlit
    Streamlit --> ConfigStore
    Streamlit --> KnowledgeStore
    Streamlit --> GeminiClient
```

## ディレクトリ構成

```
.
├── app/
│   ├── main.py              # FastAPI エントリポイント
│   ├── config.py            # 設定
│   ├── prompting.py         # プロンプト構築
│   ├── line/                # LINE 連携
│   │   ├── client.py        # Messaging API クライアント
│   │   ├── security.py      # 署名検証
│   │   └── dedup.py         # 重複排除
│   ├── llm/
│   │   └── gemini.py        # Gemini API クライアント
│   ├── rag/
│   │   ├── chroma_store.py       # ChromaDB 実装
│   │   └── firestore_knowledge_store.py  # Firestore ナレッジ実装
│   └── storage/
│       ├── config_store.py       # SQLite ロール実装
│       ├── firestore_config_store.py  # Firestore ロール実装
│       └── registry.py           # ストアファクトリ
├── streamlit_app.py         # 管理画面
├── deploy.sh                # デプロイスクリプト
├── Dockerfile.api           # FastAPI 用
├── Dockerfile.admin         # Streamlit 用
└── docs/                    # ドキュメント
```
